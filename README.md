# MEMS 3D Accelerometer Six-Position Calibration

## 项目背景

本项目是对论文 **Time- and Computation-Efficient Calibration of MEMS 3D Accelerometers and Gyroscopes** (Sensors, 2014, 14(8):14885-14915) 中提出的加速度计六位置校正方法的复现。

MEMS (Micro-Electro-Mechanical Systems) 加速度计广泛存在于智能手机、平板电脑、IoT 设备中。受制造工艺和温度影响，这类传感器存在 **比例因子误差 (Scale Factor Error)** 和 **零偏 (Bias)**。六位置静态校正法通过将设备依次放置在六个正交方位，测量每个方位下的重力分量，从而解算出比例因子和零偏，完成对原始测量值的校正。

## 校正原理

### 测量模型

加速度计的测量模型可表示为：

```
q = Cs · (a - a0)
```

其中：

- `q` 为校正后的加速度值（应接近理论重力值）
- `a` 为传感器原始测量值
- `Cs` 为比例因子校正矩阵 (scale factor correction matrix)
- `a0` 为零偏向量 (bias vector)

### 六位置预期重力分布

将设备依次放置在六个正交方位上，使每个轴的正方向和反方向交替朝上。在理想条件下（无零偏、无标度因数误差），各位置的加速度测量值应如下表所示：

| 方位 | 轴朝向 | 预期 X (m/s²) | 预期 Y (m/s²) | 预期 Z (m/s²) |
|------|--------|---------------|---------------|---------------|
| +X   | X 轴正方向朝上 | +g | 0 | 0 |
| -X   | X 轴负方向朝上 | -g | 0 | 0 |
| +Y   | Y 轴正方向朝上 | 0 | +g | 0 |
| -Y   | Y 轴负方向朝上 | 0 | -g | 0 |
| +Z   | Z 轴正方向朝上 | 0 | 0 | +g |
| -Z   | Z 轴负方向朝上 | 0 | 0 | -g |

### 核心公式

设 `A+` 和 `A-` 分别为六个方位中重力轴测量值构成的对角矩阵：

- **A+** = diag(posX, posY, posZ)  — 正轴朝上时各轴的实测值
- **A-** = diag(negX, negY, negZ)  — 负轴朝上时各轴的实测值

> **重要说明**: A+ 和 A- 是**对角矩阵 (diagonal matrix)**。每个方位只取其重力轴向的测量分量（例如 +X 方位只取 X 轴测量值，忽略 Y、Z 轴的串扰分量），而非完整的三维向量。这与代码实现 (main.py:39-42) 保持一致。

计算步骤：

1. **重力参考值**:
   ```
   g = (posX + posY + posZ - negX - negY - negZ) / 6
   ```

2. **比例因子校正矩阵**:
   ```
   Cs = 2g · (A+ − A−)⁻¹
   ```

3. **零偏向量**:
   ```
   a0 = (A+ + A−) / 2
   ```

4. **校正输出**:
   ```
   q = Cs · (a − a0)
   ```

## 数据采集

| 参数 | 值 |
|------|-----|
| 设备 | 联想拯救者 Y700 (Lenovo Legion Y700) |
| 软件 | phyphox (Physical Phone Experiment) |
| 采样率 | 100 Hz |
| 每位置采样时长 | ≈ 30 秒 |
| 数据处理段 | 中间 10 秒（第 10-20 秒），避开起止瞬态 |

采集方式是手持设备，依次将六个方位朝上静止放置，记录每段加速度数据。

## 算法说明

`main.py` 的核心流程如下：

1. **读取数据**: 使用 pandas 读取六个 CSV 文件，每文件包含时间戳和三轴加速度数据。

2. **截取稳定段**: 代码使用 `start_row=1000, end_row=2000`（第 8 行），实际通过 `df.iloc[start_row - 1 : end_row, cols]` 读取第 999-1999 行（0-indexed）。以 100 Hz 采样率换算，对应约第 9.99 秒到第 19.99 秒的数据，即 30 秒采集时长的中间 10 秒稳定段，避免了开始放置和结束时的瞬态干扰。

3. **计算各轴均值**: 对每个方位的 CSV 读取列索引 1-3（对应 X、Y、Z 轴加速度数据），在稳定段范围内取均值，得到每个方位的三轴平均测量值。

4. **提取重力轴分量**: 从每个方位的三轴均值中只提取重力轴向的分量：
   - `posX` = +X.csv 的 X 轴均值
   - `posY` = +Y.csv 的 Y 轴均值
   - `posZ` = +Z.csv 的 Z 轴均值
   - `negX` = -X.csv 的 X 轴均值
   - `negY` = -Y.csv 的 Y 轴均值
   - `negZ` = -Z.csv 的 Z 轴均值

5. **构建对角矩阵**: 将提取出的重力轴分量构造成对角矩阵 A+ 和 A-。注意：此处**不**使用完整的三维向量，而是仅用各方位重力轴方向的分量构成对角矩阵（main.py 第 39-42 行），这是本方法的核心简化。

6. **计算校准参数**: 依据校正原理章节的公式依次计算 g、Cs、a0。

7. **校正并输出**: 对六个方位的原始三轴向量应用 `q = Cs · (a − a0)`，输出校正后的结果。
